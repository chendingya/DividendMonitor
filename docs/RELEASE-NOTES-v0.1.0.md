# Release Notes v0.1.0

发布日期：2026-04-26

## 概览

`v0.1.0` 是项目首个可分发版本，完成了从股票单资产到股票、ETF、基金多资产能力的扩展，并补齐了桌面端打包链路。

本版本可交付内容包括：

- Windows 安装包
- Windows 目录版可执行程序
- 多资产搜索、详情、自选、对比、回测
- 股票估值分位与趋势分析
- 浏览器预览与桌面端统一数据链路

## 重点更新

### 1. 多资产能力

- 新增 ETF 与基金资产类型
- 搜索入口统一为资产搜索，支持结果页选择后进入详情
- 详情页、自选页、对比页、回测页统一按 `assetKey` 驱动
- 浏览器预览与桌面端共享同一套资产能力协议

### 2. 持仓与工作台

- 新增多资产持仓录入与管理
- 支持多笔买卖交易记录
- 支持按资产合并展示净持仓、净成本、平均成本
- 工作台收益展示已兼容股票与基金不同口径

### 3. 估值分析

- 补齐股票 PE / PB 分位计算
- 新增详情页估值趋势图
- 支持：
  - PE 与 PE 分位同图展示
  - PB 与 PB 分位同图展示
  - 10 年 / 20 年分位切换
  - 近 5 年 / 近 10 年 / 全部时间范围切换

### 4. 打包与交付

- 接入 `electron-builder`
- 支持：
  - `npm run dist:dir`
  - `npm run dist:win`
- 已可生成：
  - Windows 安装包
  - `win-unpacked` 目录版程序
