info 指标说明

股息率 (Dividend Yield) expand_more
股息率是一年的总派息额与当时市价的比例。计算公式：(年度每股总派息 / 当前股价) × 100%。它是衡量企业是否具有投资价值的重要标尺之一。
派息比率 (Payout Ratio) expand_more
派息比率指公司将其净利润支付给股东作为股息的比例。计算公式：(每股股息 / 每股收益) × 100%。通常健康的派息比率在30%-70%之间。
成本收益率 (Yield on Cost) expand_more
基于您最初买入股票的成本计算的股息率。计算公式：(当前年度每股总派息 / 您的平均买入成本) × 100%。随着公司股息增长，长期持有的成本收益率通常会显著上升。
